Module Descriptions

M6 KSeF Fetcher
ksef_fetcher.py
Connects to the Polish e-Invoice system (KSeF) and downloads XML invoices.
  • Authenticates using XAdES-BES certificate signing
  • Queries KSeF API for new invoices since last run
  • Downloads FA(3) XML files and parses structured invoice data directly (no LLM needed)
  • Runs first — KSeF invoices are already structured, fast to process
KSeF API
M1 Email Fetcher
email_fetcher.py
Scans Gmail inbox for emails with PDF invoice attachments.
  • Searches for unread emails matching invoice criteria
  • Downloads PDF and ZIP attachments
  • Extracts PDFs from ZIP archives
  • Passes downloaded files to the classification stage
Gmail API — gmail.modify
M2 Invoice Classifier
invoice_classifier.py
Determines if a PDF is actually an invoice and verifies seller identity.
  • Regex-based NIP (tax ID) detection in PDF text
  • OCR fallback for scanned/image PDFs
  • Matches NIP against known contractors list
  • Rejects non-invoice documents (proformas, offers, etc.)
M7 Input Sanitizer
input_sanitizer.py
Planned — defense against prompt injection in PDF content before LLM processing.
  • Sandwich prompt technique (system instructions wrap user content)
  • Regex sanitizer strips suspicious patterns
  • Safeguard model pre-checks content before main LLM
M3 Data Extractor
data_extractor.py
Uses local LLM (Ollama) to read invoice fields from PDF text.
  • Sends PDF text content to Ollama with structured extraction prompt
  • Extracts: seller, buyer, amounts, dates, invoice number, line items
  • Returns structured JSON with all 19 spreadsheet columns
  • Handles various invoice formats (Polish and international)
Local LLM — Ollama
M4 Storage Manager
storage_manager.py
Organizes invoice files on Google Drive and registers them in Google Sheets.
  • Creates monthly folders on Google Drive (e.g. "2026-03")
  • Uploads PDF/XML files with standardized naming
  • Appends invoice data row to Google Sheets register (19 columns)
  • Detects duplicates by invoice number before inserting
Drive API Sheets API
M5 Email Marker
email_marker.py
Marks processed emails in Gmail to prevent reprocessing.
  • Adds "Invoice" label to processed emails
  • Marks emails as read
  • Runs last — only after successful storage
Gmail API — gmail.modify
